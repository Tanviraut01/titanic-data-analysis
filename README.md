# Titanic Data Analysis Dashboard

## 📊 Project Overview

This project analyzes the Titanic dataset to understand survival patterns using data analysis and visualization.

## 🔧 Tools Used

* Python (Pandas, Matplotlib)
* SQL (SQLite)
* Power BI

## 📈 Key Insights

* Females had higher survival rate than males
* 1st class passengers had highest survival
* Majority passengers did not survive
